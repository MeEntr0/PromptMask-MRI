# Real Sample Assets

This folder contains a minimal authorized PNG sample for the static product demo.

Included assets:

- `raw.png`: cleaned pelvic MRI slice.
- `bladder-mask.png`: transparent bladder mask for canvas overlay.
- `bladder-overlay-preview.png`: preview image for documentation and manual QA.

These files are intended for research prototype and portfolio demonstration only. The repository does not include full training datasets, NIfTI volumes, DICOM files, or patient-identifiable metadata.
